# Truth Lens - 产品方案说明书 (v1.0)

## 1. 产品愿景
在自媒体信息爆炸、谣言横行的时代，Truth Lens 旨在成为用户浏览信息的“数字显微镜”，通过大模型实时检索、溯源与信源权重分析，提供客观的信息真伪评估及深度背景透视。

## 2. 核心功能
### 2.1 意图感应交互 (Intent-Aware Interaction)
- **划选触发**：用户选中网页中的某段文字，弹出“Lens 校验”悬浮球。
- **右键菜单**：集成至系统右键，支持“深度调查该观点”。
- **静默监测 (可选)**：针对高频传播的新闻类页面，提供低干扰的初步风险预警。

### 2.2 竞争性溯源引擎 (Competitive Fact-Checking)
- **多维度检索**：并行检索支持（Pro）与反对（Con）该断言的信息。
- **事实原子提取**：将复杂文本拆解为可校验的“事实原子”。
- **实时搜索集成**：接入 Tavily AI Search API 获取全网最新动态。

### 2.3 信源可信度矩阵 (Credibility Matrix)
- **L0 (官方/学术)**：政府机构、权威学术期刊、主流通讯社（权重 1.0）。
- **L1 (深度媒体)**：财新、南方周末、WSJ 等知名深度报道媒体（权重 0.8）。
- **L2 (大 V/机构号)**：认证后的专业垂直自媒体（权重 0.5）。
- **L3 (一般自媒体)**：个人博主、匿名营销号（权重 0.2）。

### 2.4 社区真理账本 (Truth Ledger)
- **众包纠偏**：用户可对 AI 结论提出质疑，并上传新证据。
- **信用分体系**：基于历史纠偏准确率建立用户信用等级，高信用等级用户可修正 AI 标签。

## 3. 技术架构
- **前端 (Extension)**：React + Tailwind CSS + Vite (CRXJS)。
- **逻辑编排**：LangChain (用于 RAG 工作流编排)。
- **搜索引擎**：Tavily API (专注 AI 检索的工具)。
- **后端 (API Proxy)**：用于管理 API Key、缓存搜索结果及处理社区数据库。
- **大模型**：GPT-4o (复杂推理) + GPT-4o-mini (快速预判)。

## 4. 评估逻辑
`Final_Score = Σ(信源评分 * 内容匹配度 * 时效性系数) / 归一化因子`
- 结论输出：[真实] / [存疑] / [误导] / [虚假]。
